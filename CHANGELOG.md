# Changelog

## [1.0.0] - 2026-03-03
### Added
- CloudMailin Trigger node (webhook-based, real-time)
- Manual Setup mode (no API key, works for all accounts)
- Auto-Provision mode (Management API, auto create/delete addresses)
- Simplified output format with clean email fields
- Full raw payload output mode
- Filters: sender, subject, require attachments
- Attachment support: base64 embedded + URL-stored
- SPF, TLS metadata in output
- CloudMailin API credentials (Management API + SMTP)
